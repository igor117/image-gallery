- Redux

- Immutable
- Recompose
- Redux-Saga
- React-Router
- React-Router-Redux

### Getting Started

There are two methods for getting started with this repo.

```
  > git clone https://github.com/igor117/image-gallery.git
  > cd image-gallery
  > npm install
  > npm start
  > in your Browser open http://localhost:3000
```
